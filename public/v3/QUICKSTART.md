# Web Dashboard V3 快速启动指南

## 快速开始

### 1. 启动 ClaudePulse

```bash
cd C:\Users\666\.claude\hooks\scripts\claude-pulse
node index.js
```

### 2. 访问 Dashboard

打开浏览器访问：

- **V3 增强版**: http://localhost:3000/v3/
- **原版**: http://localhost:3000/

### 3. 功能演示

#### 拖拽布局
1. 鼠标悬停在卡片上
2. 拖拽卡片到新位置
3. 调整卡片大小
4. 布局自动保存

#### 主题切换
1. 点击右上角的 🌙/☀️ 按钮
2. 切换 Light/Dark 主题
3. 主题自动保存

#### 会话搜索
1. 在会话列表的搜索框输入关键词
2. 实时过滤会话
3. 清空搜索框恢复所有会话

#### 实时通知
1. 允许浏览器通知权限
2. 新会话开始时收到通知
3. 错误发生时收到通知

## 常见问题

### Q: 为什么 Service Worker 没有注册成功？
A: Service Worker 需要 HTTPS 或 localhost 环境。如果在生产环境部署，请确保使用 HTTPS。

### Q: 为什么收不到浏览器通知？
A: 需要在浏览器中允许通知权限。首次访问时会弹出权限请求。

### Q: 布局保存在哪里？
A: 布局保存在浏览器的 localStorage 中，清除浏览器数据会丢失布局。

### Q: 如何重置布局？
A: 清除浏览器的 localStorage 或点击"布局设置"中的"重置为默认布局"按钮。

### Q: V3 版本会替代原版吗？
A: 不会。V3 是增强版，原版仍然可用。用户可以根据需求选择使用。
